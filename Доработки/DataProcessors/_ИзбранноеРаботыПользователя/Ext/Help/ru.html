Обработка расширяет возможности стандартного избранного и позволяет:

  1. Добавлять внешние файлы в избранное
  2. Группировать ссылки

Важно: При использовании обработки ссылки из стандартного избранного перемещаются в список. Это происходит при открытии обработки.
Если ссылку нужно видеть в стандартном избранном, то нужно для нее нажать кнопку

Как добавить ссылку в избранное?
Это можно сделать следующим образом:

  • нажатием на кнопку "Добавить в избранное" в диалоге, открывающемся при выполнении команды меню "Сервис — Получить ссылку"
  • выполнить команду "Сервис — Добавить в избранное" главного меню;
  • выполнить команду "Добавить в избранное" выпадающего меню, открываемого при нажатии кнопки "Избранное", расположенной в заголовке окна;
  • с помощью сочетания клавиш Ctrl + D.

Т.е. это делает как обычно при работе со стандартным избранным.

Общие внешние объекты

Список внешних файлов можно сохранить и использовать в избранном разных конфигураций.
Для этого необходимо открыть настройки и добавить (создать) списки общих внешних объектов.
